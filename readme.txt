详细：	已将预处理代码集成到了main.py中，仅供参考
	包含二值化，边角处理，腐蚀膨胀，滤波等形态学处理
	重点函数"rotate_bound()"可用于原始图片的旋转，旋转结果可以直接放入main.py中进行分割

5.visual文件夹		几张临时效果图片
